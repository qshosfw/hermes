# <Shuffle className="inline w-6 h-6 mr-2 text-rose-400" /> 2. PN15 Whitening

To maintain a balanced DC offset and prevent bit-slip in the receiver's FSK decoder, all packets are scrambled using a pseudo-random noise sequence before transmission.

## 2.1 The Scrambling Logic

Whitening is applied as a bitwise XOR operation between the raw 96-byte packet and a sequence generated by a **PN15 Linear-Feedback Shift Register (LFSR)**.

## 2.2 LFSR Polynomial

Hermes uses the standard PN15 polynomial:
```math
G(x) = x^{15} + x^{14} + 1
```

## 2.3 C Implementation (Pseudocode)

The LFSR is seeded with the lower 16 bits of the **Sync Word** to ensure per-network uniqueness if the sync word is changed, while remaining deterministic for all participants.

```c
/**
 * @brief Scrambles/Descrambles 96 bytes using PN15 LFSR.
 * @param buffer data to whiten (in-place)
 * @param sync_word 32-bit sync word used as seed
 */
void Hermes_Whiten(uint8_t* buffer, uint32_t sync_word) {
    // Seed LFSR (cannot be 0)
    uint16_t lfsr = (uint16_t)(sync_word & 0xFFFF);
    if (lfsr == 0) lfsr = 0xACE1; 
    
    for (int i = 0; i < 96; i++) {
        uint8_t mask = 0;
        // Generate mask byte (8 bits)
        for (int b = 0; b < 8; b++) {
            // Polynomial: Bit 0 XOR Bit 1
            uint16_t new_bit = ((lfsr >> 0) ^ (lfsr >> 1)) & 1;
            lfsr = (lfsr >> 1) | (new_bit << 14);
            mask = (mask << 1) | (uint8_t)new_bit;
        }
        buffer[i] ^= mask;
    }
}
```

## 2.4 Why Whiten?

1. **DC Balance**: Prevents long strings of `0`s or `1`s from biasing the FM discriminator.
2. **Transition Density**: Increases the number of zero-crossings, which helps the receiver's clock recovery loop stay locked.
3. **Spectral Spreading**: Distributes RF energy more evenly across the occupied bandwidth.

> [!WARNING]
> Whitening must be applied **BEFORE** Reed-Solomon encoding in the transmission pipeline. This ensures that the parity bytes are calculated over the scrambled stream, localized bit errors are NOT multiplied by the descrambling process.
