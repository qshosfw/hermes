# Advanced Mesh Simulator

This immersive playground visually establishes the mathematical algorithms behind the Hermes Protocol's **SNR-based Controlled Flooding**. 

Nodes geometrically calculate pseudo-random Time-to-Live rebroadcast delays weighted heavily by their receiving Signal-To-Noise Ratios (SNR). This explicitly prioritizes and guarantees that nodes with robust antenna backbone conditions transmit earliest, cleanly suppressing weaker outlier nodes and building lightning-fast routing corridors across the physical RF mesh without explicit acknowledgement handshakes.
