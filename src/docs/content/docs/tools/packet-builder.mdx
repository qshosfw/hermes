# Interactive Packet Builder

The Packet Builder is a comprehensive decoding and encoding diagnostic tool. It allows developers to manually construct full Hermes Protocol frames layer-by-layer, adjusting addressing flags, payload content, and cryptographic parameters in real-time.

Use this tool to generate mock hex-strings for embedding in your own firmware unit tests or to visualize exactly how the bits are packed over the air.
